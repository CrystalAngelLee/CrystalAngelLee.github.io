命令注册​

init 命令注册​

通过 npm init egg --type=simple 初始化 egg 项目

npm init 默认会把当前目录作为 npm 项目做初始化，如果在后面加上一个命令 (例如 egg) npm 就会帮我们执行 egg 命令

当使用 npm init 后面加上一个项目或者脚手架名称的时候 会调用 create -后面这个名称的NPM 包 「npm init egg 实际上执行的是 create-egg 脚手架」
